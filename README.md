# Instagram Reel Automation (Python)

This project automatically:

- Reads daily content from Google Sheets
- Adds styled text overlay to a video
- Mixes background music
- Exports a final Reel video
- Uploads it to Instagram using API

---

## Features

- Google Sheet based content scheduling
- Dynamic text wrapping & styling
- Rounded background with shadow
- Background music mixing
- Instagram auto-upload
- Environment variable security
